🍕 Pizza Sales Business Intelligence Dashboard
📌 Project Overview

This project analyzes 48,000+ pizza sales records to uncover key business insights using Python, SQL, and Power BI.
An interactive Power BI dashboard was developed to track revenue trends, customer ordering patterns, and top-performing products.

The goal of this project is to demonstrate data analysis, business intelligence, and visualization skills by transforming raw sales data into actionable insights.
